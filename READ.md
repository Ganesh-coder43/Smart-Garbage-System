# 🗑 Smart Garbage Management System

## 📌 Project Overview
This website helps users report garbage issues in their area by uploading images and tracking complaints. It is designed to improve cleanliness and support smart city solutions using AI-based features.

## 🎯 Objectives
- Report garbage issues with image upload
- Track complaint status
- Improve waste management efficiency
- Support smart city cleanliness

## 🛠 Tools Used
- HTML
- CSS
- JavaScript
- AI Tools
